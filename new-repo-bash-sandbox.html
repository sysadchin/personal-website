New Repo: Bash Sandbox

Date Wed 02 March 2016 Tags projects

One of the biggest frustrations when writing scripts is when environment variables become stale between iterations. This happens a lot during debugging and can have unintended results.

I've put together a shell script that cleans the environment, sources a variable file, then executes your script. When either the script or envvar file are modified, it reiterates again. This saves a tremendous amount of time in the development workflow.

You can find it on github.
